Description

Worksuite SAAS is a project management software written in Laravel 8 (PHP Framework) which is specifically developed for freelancers and SMEs (Small/Medium sized enterprises). You can manage your company's daily work, your employee's tasks, keep a track on project's progress and much more. It is designed with latest security and code standards.

Server Requirements

The Server needs to meet the following requirements to run the application:

  • PHP >= 8.2.X
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • GD PHP Extension
  • Fileinfo PHP Extension
